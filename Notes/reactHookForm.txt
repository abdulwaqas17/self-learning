## âœ… **3. Ye line kya kar rahi hai?**

```js

  register,
  handleSubmit,
  formState: { errors },

```

### ğŸ“Œ Breakdown:

| Keyword            | Kya karta hai                                                                         |
| ------------------ | ------------------------------------------------------------------------------------- |
| `useForm()`        | React Hook Form ka main hook hai. Yeh form manage karta hai.                          |
| `register()`       | Har input field ko register karta hai taake uska data React Hook Form track kare.     |
| `handleSubmit()`   | Form submit handler wrap karta hai (validation ke saath).                             |
| `formState.errors` | Validation errors ko store karta hai. Aap isse conditionally messages dikha sakte ho. |
